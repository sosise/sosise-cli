## Installation
Install `sosise-cli` via `npm` or `yarn` globally.

```sh
npm i sosise-cli -g
```

## Usage
Use `new` to generate your project.

```sh
sosise new <name>
```

## Build
```sh
git clone https://github.com/sosise/sosise-cli.git sosise-cli
cd sosise-cli
npm install
npm run build
./bin/sosise --help
```
